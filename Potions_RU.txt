Базовые зелья:
Колба с водой + Нарост Нижнего мира = Грубое зелье  
Колба с водой + Редстоуновая пыль = Непримечательное зелье  
Колба с водой + Светопыль = Густое зелье  
Колба с водой + Приготовленный паучий глаз = Зелье слабости (1:30)  

Зелья с положительными эффектами:
Грубое зелье + Сверкающий ломтик арбуза = Зелье исцеления (Мгновенное здоровье I)  
Зелье исцеления (Мгновенное здоровье I) + Светопыль = Зелье исцеления (Мгновенное здоровье II)  
Грубое зелье + Лавовый крем = Зелье огнестойкости (3:00)  
Зелье огнестойкости (3:00) + Редстоуновая пыль = Зелье огнестойкости (8:00)  
Грубое зелье + Слеза гаста = Зелье регенерации (0:45)  
Зелье регенерации (0:45) + Редстоуновая пыль = Зелье регенерации (1:30)  
Зелье регенерации (0:45) + Светопыль = Зелье регенерации II (0:22)  
Грубое зелье + Огненный порошок = Зелье силы (3:00)  
Зелье силы (3:00) + Редстоуновая пыль = Зелье силы (8:00)  
Зелье силы (3:00) + Светопыль = Зелье силы II (1:30)  
Грубое зелье + Сахар = Зелье стремительности (3:00)  
Зелье стремительности (3:00) + Редстоуновая пыль = Зелье стремительности (8:00)  
Зелье стремительности (3:00) + Светопыль = Зелье стремительности II (1:30)  
Грубое зелье + Золотая морковь = Зелье ночного зрения (3:00)  
Зелье ночного зрения (3:00) + Редстоуновая пыль = Зелье ночного зрения (8:00)  
Зелье ночного зрения (3:00) + Приготовленный паучий глаз = Зелье невидимости (3:00)  
Зелье невидимости (3:00) + Редстоуновая пыль = Зелье невидимости (8:00)  
Грубое зелье + Иглобрюх = Зелье подводного дыхания (3:00)  
Зелье подводного дыхания (3:00) + Редстоуновая пыль = Зелье подводного дыхания (8:00)  
Грубое зелье + Кроличья лапка = Зелье прыгучести (3:00)  
Зелье прыгучести (3:00) + Редстоуновая пыль = Зелье прыгучести (8:00)  
Зелье прыгучести (3:00) + Светопыль = Зелье прыгучести II (1:30)  
Грубое зелье + Мембрана фантома = Зелье медленного падения (1:30)  
Зелье медленного падения (1:30) + Редстоуновая пыль = Зелье медленного падения (4:00)  
Грубое зелье + Блок слизи = Зелье слизистости (3:00)  
Грубое зелье + Паутина = Зелье плетения (3:00)  
Грубое зелье + Камень = Зелье заражения (3:00)  
Грубое зелье + Стержень вихря = Зелье ветрового заряда (3:00)  

Зелья с отрицательными эффектами:
Грубое зелье + Паучий глаз = Зелье отравления (0:45)  
Зелье отравления (0:45) + Редстоуновая пыль = Зелье отравления (1:30)  
Зелье отравления (0:45) + Светопыль = Зелье отравления II (0:21)  
Зелье исцеления (Мгновенное здоровье I) + Приготовленный паучий глаз = Зелье урона (Мгновенный урон I)  
Зелье отравления (0:45) + Приготовленный паучий глаз = Зелье урона (Мгновенный урон I)  
Зелье урона (Мгновенный урон I) + Светопыль = Зелье урона (Мгновенный урон II)  
Зелье стремительности (3:00) + Приготовленный паучий глаз = Зелье замедления (1:30)  
Зелье прыгучести (3:00) + Приготовленный паучий глаз = Зелье замедления (1:30)  
Зелье замедления (1:30) + Редстоуновая пыль = Зелье замедления (4:00)  
Зелье замедления (1:30) + Светопыль = Зелье замедления IV (0:20)  
Зелье слабости (1:30) + Редстоуновая пыль = Зелье слабости (4:00)  

Смешанные зелья:
Грубое зелье + Черепаший панцирь = Зелье черепашьей мощи (0:20) (Замедление IV + Сопротивление III)  
Зелье черепашьей мощи (0:20) + Редстоуновая пыль = Зелье черепашьей мощи (0:40) (Замедление IV + Сопротивление III)  
Зелье черепашьей мощи (0:20) + Светопыль = Зелье черепашьей мощи II (0:20) (Замедление VI + Сопротивление IV)  

Взрывные зелья:
Любое питьевое зелье + Порох = Взрывное зелье [Эффекта]  
Колба с водой + Порох = Взрывная колба с водой  
Грубое зелье + Порох = Взрывное зелье (без эффекта)  
Непримечательное зелье + Порох = Взрывное зелье (без эффекта)  
Густое зелье + Порох = Взрывное зелье (без эффекта)  

Оседающие зелья:
Любое взрывное зелье + Драконье дыхание = Оседающее зелье [Эффекта]  
Взрывная колба с водой + Драконье дыхание = Оседающая колба с водой  

Примечание: Времена указаны для Java Edition, если не указано иное (в Bedrock Edition некоторые могут отличаться). Новые зелья из 1.21 (Слизистости, Плетения, Заражения, Ветрового заряда) не имеют усиленных или продлённых версий.
